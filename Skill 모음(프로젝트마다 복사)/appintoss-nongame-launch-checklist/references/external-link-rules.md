# 자사 앱 설치/외부 링크 가이드라인 상세

앱인토스 미니앱의 자사 앱 설치 유도 및 외부 링크 이동 제한 정책이다.
위반 시 콘텐츠 비노출 또는 서비스 이용 제재가 적용될 수 있다.

출처: https://developers-apps-in-toss.toss.im/checklist/miniapp-external-link.md

---

## 1. 자사 앱 설치 유도 금지

앱인토스 미니앱은 "앱 설치 없는 간편한 서비스 이용"이 핵심 가치다.

### 제한되는 행위

- 앱 설치를 직접적으로 유도하는 문구
  - 예: "앱을 설치하시면 더 많은 기능을 이용할 수 있어요."
  - 예: "전용 쿠폰을 받으시려면 자사 앱을 설치하세요."
  - 예: "앱 다운로드하고 첫 구매하면 할인 받을 수 있어요."
- 앱 설치를 유도하는 이미지 또는 배너 삽입
- 앱 마켓 링크(App Store, Play Store) 삽입
- 자사 앱 설치와 관련된 혜택 안내
- 그 외 자사 앱 설치 유도로 판단될 수 있는 콘텐츠 및 기능

---

## 2. 외부 링크 이동 제한

사용자가 토스 미니앱 환경을 벗어나 외부 웹사이트나 앱으로 이동하게 만드는 행위를 제한한다.

### 제한되는 행위

- 앱 내 기능으로 등록한 서비스를 미니앱 내에서 모두 경험할 수 없는 형태
- 외부 결제창으로의 이동
- 앱 다운로드 페이지 또는 설치를 위한 외부 페이지 연결
- 공유하기 기능의 링크가 자사 웹사이트로 랜딩되는 상태
- 미니앱의 주요 기능이나 흐름을 외부 링크에 의존하는 구조
- 그 외 주요 기능이 외부 링크에서 실행되도록 설계된 경우

### 허용되는 외부 링크

- 법률상 고지 및 필수 안내 목적의 외부 링크
- 공공기관 또는 제휴기관의 공식 페이지 연결
- 단순 정보 확인을 위해 자사가 아닌, 타사의 웹사이트로 이동
- 미니앱 기능 내에서 완결되지 않는 일부 특수 상황:
  - 각 제품을 소개/추천하고, 최저가로 구매할 수 있는 플랫폼으로 이동하는 서비스
  - 혜택을 제공하기 위해 각 쿠폰을 발급하는 플랫폼으로 이동하는 서비스

---

## 3. 기존 앱 서비스와의 관계

기존 앱의 모든 서비스를 미니앱으로 기능화하지 않아도 된다.

**핵심 원칙**: 콘솔에서 설정한 '앱 내 기능'의 기능을 미니앱 내에서 모두 경험할 수 있으면 된다.

- "~견적내기" 기능 → 미니앱 내에서 견적을 완결성 있게 낼 수 있으면 OK
- "~견적내고 결제하기" 기능 → 견적부터 결제까지 미니앱 내에서 진행해야 함
- 기능을 최소화하여 먼저 출시, 추후 업데이트도 가능

---

## 4. 위반 시 제재

1. 관련 내용 고지 및 콘텐츠 수정 요청
2. 토스 앱 내 미노출 처리
3. 반복 위반 시 미니앱 서비스 등록 또는 운영 제한

```
외부 링크 정책 점검:
- [ ] 앱 설치 유도 문구가 없는가?
- [ ] 앱 설치 유도 이미지/배너가 없는가?
- [ ] 앱 마켓 링크가 없는가?
- [ ] 앱 설치 관련 혜택 안내가 없는가?
- [ ] 콘솔 등록 기능이 모두 미니앱 내에서 완결되는가?
- [ ] 외부 결제창으로 이동하지 않는가?
- [ ] 공유 링크가 자사 웹사이트가 아닌 토스 공유 링크인가?
- [ ] 주요 기능 흐름이 외부 링크에 의존하지 않는가?
- [ ] 사용 중인 외부 링크가 허용 범위에 해당하는가?
```
