# 아이데이션 프로세스 & 트러블슈팅

소재 발상법과 성과 안 나올 때의 문제 진단/해결 기법.

---

## 5단계 아이데이션 프로세스

핵심: "머릿속이 아니라 화면에서 아이데이션 하기"

### STEP 1 — 마인드 리셋

서비스에 대해 아는 모든 걸 잊는다. 처음 보는 사람 입장에서 생각한다.
"우리 서비스의 장점, 핵심 기능, 모든 것을 잊으세요."

### STEP 2 — 화면 감상

앱의 핵심 화면에 보이는 모든 요소를 관찰하고 적는다:
텍스트, 숫자, 버튼, 이미지, 캐릭터, 색상, 재화명 등 모든 것.

예시 (캣워즈 게임):
- "고양이를 소환하면 보석을 받을 수 있네"
- "고양이는 플레이어이고 보석은 재화겠구나"
- "고양이들이 작고 귀엽네, 모자와 칼을 들고 있어 전사 같네"
- "적의 타워를 무너뜨려야 하네"
- "광고를 보면 붕어빵을 얻을 수 있네"
- "붕어빵 3개 = 고양이 소환"

### STEP 3 — 발산

관찰한 것들을 조합해서 가능한 모든 메시지를 만든다. 질보다 양.

예시 (캣워즈):
- 감상 "고양이를 소환하면 보석을 받을 수 있네" → "게임 보석 받기 / 고양이 8마리 소환하면 받아요."
- 감상 "붕어빵 3개 = 고양이 소환" → "고양이 1마리 소환하는 법 / 게임에서 붕어빵 3개 모아요."
- 감상 "고양이들이 귀엽고 적 타워를 무너뜨려야" → "적군의 성을 무너뜨려요 / 게임에서 작은 고양이 전사와 함께해요."
- 감상 "광고 보면 붕어빵 얻음" → "광고 보고 붕어빵 받기 / 게임에서 붕어빵 모으고 고양이 소환해요."

### STEP 4 — 수렴

4가지 원칙으로 평가. "잘 읽히는 것, 내가 누르고 싶은 것"만 고른다.

평가 기준:
- "광고" 단어가 들어가면 부정적으로 느껴질 수 있다 → 탈락
- "그 1마리를 빨리 보고 싶다"는 호기심이 든다 → 후보
- "사람들이 보석 좋아하겠지?" → 후보
- "빨리 가서 싸우는 거 보고 싶다" → 후보

주관적 느낌도 중요하다. 본인이 누르고 싶지 않은 문구는 탈락.

### STEP 5 — 더하기

선택한 메시지를 다듬고 발전시킨다. 키워드를 더 구체적이고 매력적으로.

- "자연" → "풀숲", "고양이 전사가 기다려요"
- "고양이 1마리" → "전투 모자 쓴 고양이"
- "게임 보석" → "보석 30개 받기" (숫자 구체화)

## 문제 진단 & 해결 프로세스

생성한 소재의 성과가 안 나오거나 막혔을 때 사용.

### 5가지 자가 체크

1. 하나의 메시지만 담았나?
2. 너무 복잡하지는 않나?
3. 충분히 구체적인가?
4. 확실한 보상인가?
5. 더 쉽게 풀어쓸 수 없을까?

### 문제: 너무 복잡하다 → 키워드 분해 테크닉

복잡한 문구를 키워드 단위로 쪼개고, 꼭 필요한 2~3개만 골라 재조합한다.

실전 예시:
1. 원본: "지자체별 전기차 혜택 비교 분석 / 결과를 확인해보세요."
2. 키워드 분해: 지자체별 / 전기차 / 혜택 / 비교 / 분석 / 결과
3. 핵심 2~3개 선택: **전기차**, **혜택**
4. 재조합 1차: "전기차 혜택을 비교해요 / 토스에서 확인해보세요."
5. 재조합 2차: "전기차 타면 받는 지원금 / 토스에서 확인해보세요."
6. 키워드 추가: "우리 지역에서 주는 전기차 지원금 / 토스에서 확인해보세요."

### 문제: 충분히 구체적이지 않다 → 화면에서 행동 찾기

앱 화면을 다시 보고 유저의 구체적 행동을 찾아 단계적으로 구체화한다.

실전 예시 (전기차 보조금 서비스):
1. 화면 관찰: "시/도를 선택해요" 드롭다운 → "지역을 선택하면 보조금 확인 가능"
2. 1차 구체화: "전기차 보조금 확인해요 / 내 지역 선택하면 볼 수 있어요."
3. 2차 구체화: "전기차 보조금 확인해요 / 지역 **1개** 고르면 볼 수 있어요." (숫자)
4. 3차 구체화: "전기차 보조금 확인해요 / 내 지역 **누르면** 볼 수 있어요." (쉬운 동사)

### 문제: 더 쉽게 풀어쓸 수 없을까 → 소리 내서 읽기 테스트

문구를 소리 내서 읽고, 어색한 단어를 더 자주 쓰는 표현으로 교체한다.

실전 예시:
1. "전기차 혜택을 비교해요 / 내 지역 선택하면 볼 수 있어요."
2. "내 지역" → "동네"가 더 자연스럽다 → "내 **동네** 누르면"
3. "전기차 보조금" → "보조금"이 어렵다 → "전기차타면 **정부에서 주는 돈**"
4. 최종: "전기차타면 정부에서 주는 돈 / 토스에서 확인해보세요."
